# Pixel Media Quest

เกม Pixel Art เพื่อการเรียนรู้ **Graphic Design Workflow** และ **3P Production** ผ่านการสำรวจ คุยกับ NPC ค้นหาหลักฐาน และทำภารกิจทั้ง 8 ขั้นตอน

## เปิดเล่น

https://teachthatsp99-cmyk.github.io/pixel-media-quest/

## การควบคุม

- คอมพิวเตอร์: กดค้าง `WASD` หรือปุ่มลูกศร และกดตำแหน่งปุ่ม `E` เพื่อโต้ตอบ
- มือถือ: ลากจอยเสมือนเพื่อเดิน และกดปุ่ม **ใช้** เพื่อโต้ตอบ
- รองรับแป้นพิมพ์ภาษาไทยและภาษาอังกฤษ

ระบบจะบันทึกความคืบหน้าไว้ในเบราว์เซอร์ของผู้เล่นโดยอัตโนมัติ
